👴📱 Saathi — The Unified Digital Bridge for Elder Inclusion
📌 Project Overview
The platform helps senior citizens confidently use digital services such as UMANG, WhatsApp, UPI, banking apps, healthcare services, and government portals without requiring technical knowledge.

🚀 Application Workflow
User opens any Android application.
Saathi observes the screen using Accessibility Service.
User gives a voice command.
AI understands the user's intent.
Required UI element is identified.
Halo Overlay highlights the correct button.
Voice guidance explains each step.
Safety engine checks for suspicious actions.
Task is completed securely.
🎯 Problem Statement
Many elderly users struggle with:

Complex smartphone interfaces
Small buttons and text
Language barriers
Digital scams
Multi-step government services
Online banking confusion
Lack of digital confidence
Saathi bridges this digital accessibility gap by providing an intelligent, multilingual, and privacy-focused assistance layer.
